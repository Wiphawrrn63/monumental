Welcome to Monumental, the tool that allows you to explore built heritage and learn more about monuments.

Enter the name of a monument in the search box above (for example, 'Empire State Building') to learn more or explore heritage by entering a geographic region (for example, 'Berlin'). You can also click an example monument or city below.

The tool displays information and multimedia about cultural heritage monuments gathered through Wikidata, Wikipedia and Wikimedia Commons. It is created as a part of the Connected Open Heritage project by Wikimedia Sweden and Wiki Loves Monuments initiative.
